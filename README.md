# SecurityLogPlatform

SecurityLogPlatform is a web-based cybersecurity application developed using **Python Flask** that simulates the workflow of a **Security Operations Center (SOC)**. The platform enables security analysts to upload and analyze log files, classify security events based on severity, manage incident response workflows, visualize security metrics through an interactive dashboard, and generate professional PDF reports.

## Features

* Secure user authentication (Login/Logout)
* Upload and analyze security log files
* Automatic log parsing and severity classification
* Incident creation and lifecycle management
* Interactive SOC dashboard with charts and statistics
* SQLite database integration using SQLAlchemy
* PDF report generation using ReportLab
* Responsive web interface built with HTML, CSS, JavaScript, and Bootstrap

## Technology Stack

* Python
* Flask
* SQLAlchemy
* SQLite
* HTML5
* CSS3
* JavaScript
* Bootstrap 5
* Chart.js
* ReportLab

## Project Purpose

The project demonstrates the practical implementation of cybersecurity concepts such as security log analysis, incident response management, and dashboard visualization. It was developed as a **Bachelor of Computer Applications (BCA) Final Year Major Project** to simulate the core functionalities of a Security Operations Center (SOC).
